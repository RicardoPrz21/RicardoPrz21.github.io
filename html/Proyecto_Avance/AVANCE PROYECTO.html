CapacitaT

OBJETIVOS

Registro de Capacitadores y Empresas Clientes:
Implementar un sistema para que la empresa pueda agregar y gestionar información de sus capacitadores. Desarrollar un formulario para que las empresas clientes ingresen sus datos y solicitudes de capacitación.
Gestión de Cursos y Asignación de Capacitadores:
Crear un módulo para registrar cursos con costo, horarios y capacitadores asignados. Implementar un algoritmo para asignar óptimamente capacitadores a sesiones de capacitación.
Generación de Planes de Capacitación y Facturación:
Desarrollar funcionalidades para generar planes de capacitación y facturas para empresas clientes. Implementar un sistema de facturación automática.
Gestión de Pagos Mensuales a Capacitadores:
Crear un sistema que calcule automáticamente los pagos mensuales a capacitadores. Permitir a la empresa y capacitadores consultar los pagos mensuales y su historial.
Administración de Datos:
Implementar opciones para gestionar datos de empresas clientes y capacitadores. Desarrollar una funcionalidad de consulta para la empresa capacitadora sobre los capacitadores y sus asignaciones.
Diseño y Experiencia de Usuario:
Realizar una planificación para definir el diseño visual y paletas de colores. Asegurar una interfaz atractiva y fácil de usar para todos los usuarios.
